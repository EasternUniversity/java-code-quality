This script formats Java code according to Google's format style.

Usage:
    format <optional arguments> [Java file]

Arguments:
    -h
      Show this help panel.

    -r
      Recursively format all Java files in a directory.

    -s
      Silent mode: don't list the names of formatted/skipped files.
