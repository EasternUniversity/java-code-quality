Display linting information for Java code according to Google's format style.

Usage:
    lint <optional arguments> [Java file]

Arguments:
    -h
      Show this help panel.

    -f
      Force linting files without a .java extension.

    -r
      Recursively lint all Java files in a directory.

    -v
      Display version information.
